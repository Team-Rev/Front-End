# Front-end
React 와 Redux로 만든 캡스톤 디자인 프론트엔드 파트입니다.

## 실행 설명
1) Node.js 설치
https://nodejs.org/ko/
2) Create React App 설치
```
  npm install -g create-react-app
```  
3) Create React App 이용해 개발환경 구축
```
  cd [생성한 경로]
  create-react-app .
```
4) React 웹엡 실행
```
  npm run start / npm start
```
5) 배포
```
  npm run build
```
